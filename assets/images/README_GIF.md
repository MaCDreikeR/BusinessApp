
---

## 🎨 **Para adicionar GIF na tela de boas-vindas:**

### **Coloque seu GIF aqui:**
```
assets/images/welcome-animation.gif
```

### **Especificações:**
- ✅ Formato: `.gif`
- ✅ Tamanho máximo: 5 MB
- ✅ Dimensões: 800x600 ou maior
- ✅ Transparente ou fundo sólido

---

## 🔗 **Sites para GIFs:**

### **Grátis:**
- https://giphy.com/

- https://lottiefiles.com/

### **Otimizar GIF:**
